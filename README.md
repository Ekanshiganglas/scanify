# QR Code Tool

A simple Python tool to generate and scan QR codes.

## Features
- Generate QR codes from any text or URL
- Scan QR codes from image files

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python qr_tool.py
```

Then choose from the menu:
- `1` Generate QR code
- `2` Scan QR code
- `3` Exit
